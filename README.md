Terca Livre - Backend

## Requisitos
- Docker
- Docker Compose
- Java 11+
- Maven

## Como usar
- `git clone git@github.com:tercalivre/tercalivre-api.git`
- `cd tercalivre-api`
- `sudo docker-compose up -d`
